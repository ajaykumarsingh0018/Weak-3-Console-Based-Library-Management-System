# 📚 Console-Based Library Management System

A **Java-based command-line Library Management System** that allows librarians to manage books, members, and borrowing operations efficiently.
The project demonstrates **Object-Oriented Programming (OOP), File Handling, Exception Handling, and Data Structures** in Java.

---

# 🚀 Features

### 📖 Book Management

* Add new books to the library
* Remove books from the collection
* Display available books
* Search books by **ISBN, Title, or Author**

### 👤 Member Management

* Register new library members
* Track borrowed books for each member
* Maintain member records

### 🔄 Borrowing System

* Borrow books with availability checking
* Return books and update status
* Track borrowed books for each member

### 💾 Data Persistence

* Store books in **books.txt**
* Store members in **members.txt**
* Automatically load saved data when the program starts

### ⚠️ Error Handling

* Input validation
* File read/write exception handling
* User-friendly console messages

### 📊 Advanced Features

* Overdue fine calculation
* Library statistics
* Export library data to **CSV**
* Book reservation system

# ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```
git clone https://github.com/ajaykumarsingh0018/Weak-3-Console-Based-Library-Management-System.git
```

### 2️⃣ Navigate to the project folder

```
cd Console-Based-Library-Management-System.
```

### 3️⃣ Compile the Java files

```
javac *.java
```

### 4️⃣ Run the program

```
java Main
```
